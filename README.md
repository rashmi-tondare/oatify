A slot-machine-style single-page app that randomly picks from 10 curated overnight oats recipes. Spin the reels, discover your next breakfast!

## Overview

Oatify removes the daily "what should I make?" decision from your overnight oats routine. Four emoji reels — **Liquid**, **Fruit**, **Crunch**, and **Oomph** — spin and land on a cohesive recipe, revealing a full ingredient card below. No accounts, no APIs, no frameworks. Just press **Spin** and start prepping.

### Features

- **Slot machine reels** — four emoji reels spin simultaneously and land in a staggered sequence
- **10 curated recipes** — from Tropical Sunrise to Espresso Buzz
- **Full recipe card** — ingredients grouped by category appear after every spin
- **No-repeat guard** — consecutive spins always return a different recipe
- **Fully responsive** — 4-column reels on desktop, 2×2 grid on mobile
- **Zero dependencies** — no frameworks, no CDNs, no API calls
- **Accessible** — reduced-motion support, semantic HTML, ARIA attributes
